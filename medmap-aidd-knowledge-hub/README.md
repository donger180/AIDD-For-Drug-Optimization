# MedMap AIDD Bilingual Knowledge Hub

This repository is a bilingual GitHub Pages knowledge hub for MedMap AIDD.

本仓库是 MedMap AIDD 中英文双语 GitHub Pages 知识导航站。

## Purpose / 用途

Help pharmaceutical companies, biotech teams and CRO partners understand how MedMap uses AIDD, AI drug discovery, closed-loop DMTA, ADMET-AI, docking, PK/PD simulation and project knowledge bases to improve drug discovery and preclinical research efficiency.

帮助药企、生物科技公司与 CRO 合作伙伴了解 MedMap 如何通过 AIDD、AI 药物研发、闭环 DMTA、ADMET-AI、分子对接、PK/PD 模拟与项目知识库提升药物发现与临床前研究效率。

## Deployment / 部署方法

1. Create a new GitHub repository, for example `medmap-aidd-knowledge-hub`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` branch and `/root` folder.
6. Save and wait for GitHub Pages to publish.

中文：

1. 在 GitHub 新建仓库，例如 `medmap-aidd-knowledge-hub`。
2. 将本文件夹所有文件上传到仓库根目录。
3. 进入 **Settings → Pages**。
4. Source 选择 **Deploy from a branch**。
5. Branch 选择 `main`，Folder 选择 `/root`。
6. 保存后等待 GitHub Pages 发布。
